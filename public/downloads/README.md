# Event Downloads Directory

Place event PPT/PDF files here for participants to download.

## Required Files:

### Ignitron (Technical Events):
- `tech-hackathon.pptx` - Hackathon details, rules, judging criteria
- `code-combat.pptx` - Coding competition guidelines
- `circuit-mastery.pptx` - Electronics event details

### Kritansh (Cultural Events):
- `dance-battle.pptx` - Dance competition rules and themes
- `battle-of-bands.pptx` - Music competition guidelines
- `fashion-show.pptx` - Fashion show themes and rules

### Chrysalis (Gaming Events):
- `bgmi-tournament.pptx` - BGMI tournament brackets and rules
- `fifa-championship.pptx` - FIFA competition details
- `chess-championship.pptx` - Chess tournament format

## File Requirements:
- Format: PPTX or PDF
- Size: Keep under 10MB for faster downloads
- Include: Rules, schedule, prizes, contact info
- Professional design matching fest theme

## What to Include in PPTs:
1. Event name and description
2. Detailed rules and regulations
3. Judging criteria
4. Prize distribution
5. Schedule and timing
6. Venue details
7. Contact coordinators
8. Do's and Don'ts
9. Registration process
10. FAQ section

You can create these files or participants will get a 404 error when trying to download.

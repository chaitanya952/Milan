# UPI QR Codes Directory

Place your UPI payment QR codes here.

## Required Files:

1. `ignitron-upi.png` - QR code for Ignitron (Technical) events
2. `kritansh-upi.png` - QR code for Kritansh (Cultural) events  
3. `chrysalis-upi.png` - QR code for Chrysalis (Gaming) events

## QR Code Requirements:
- Format: PNG
- Size: 300x300 pixels minimum
- Must be valid UPI QR codes
- Should include your UPI ID/VPA

## How to Generate UPI QR Code:
1. Use Google Pay, PhonePe, or Paytm QR code generator
2. Or use online tools like https://www.qr-code-generator.com/
3. Test the QR code before uploading
4. Ensure it contains your correct UPI ID

## Different QR Codes:
Each event category can have a separate UPI account/QR code for better tracking and accounting.
